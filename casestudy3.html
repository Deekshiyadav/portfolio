Project Overview

The Smart City Crime Detection system leverages deep learning to analyze real-time surveillance footage and detect suspicious activities. Using a hybrid CNN-RNN model, it effectively captures both spatial and temporal features from video input. The system is integrated with a Django-based web interface that allows authorities to monitor incidents, receive alerts, and take timely action. This project aims to enhance urban safety and support smart city infrastructure through intelligent automation.
